# Отгрузка TSD

Android-приложение для терминала сбора данных Mindeo M40B. Автоматизирует создание отгрузок в МойСклад при сканировании штрихкодов заказов.

## Как работает

1. При запуске загружает кэш заказов за последние 5 дней из МойСклад (сохраняется локально — при повторном открытии тот же день кэш читается с диска без запроса к API)
2. Оператор сканирует штрихкоды заказов — каждый проверяется по кэшу мгновенно:
   - **ГОТОВО К ОТГРУЗКЕ** — заказ найден, отгрузки нет
   - **УЖЕ ОТГРУЖЕНА** — отгрузка уже существует
   - **НЕ НАЙДЕН** — заказ не найден в системе
3. Нажимает «Создать отгрузки» — приложение пакетно создаёт отгрузки для всех готовых заказов через API МойСклад

## Настройка

Создайте файл `local.properties` в корне проекта (рядом с `build.gradle`) и добавьте токен:

```
sdk.dir=/path/to/android/sdk
MOYSKLAD_TOKEN=ваш_токен_мойсклад
```

Файл `local.properties` не попадает в репозиторий — он в `.gitignore`.

## Сборка

```bash
./gradlew assembleDebug
# APK: app/build/outputs/apk/debug/otgruzka-tsd.apk
```

## Стек

- Kotlin, Retrofit 2, OkHttp, Gson, Kotlin Coroutines
- API: [МойСклад JSON API 1.2](https://dev.moysklad.ru/doc/api/remap/1.2/)
- Min SDK: 26 (Android 8.0)
